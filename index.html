Bish-Bosh 2.0

Play the Bish-Bosh

Bish-Bosh is a simple number game where you list numbers from 1 to a certain maximum number, but you replace some numbers with special words.

  • 👉 If a number is evenly divisible by 3, it's replaced with "Bish".
  • 👉 If a number is evenly divisible by 4, it's replaced with "Bosh".
  • 👉 If a number is evenly divisible by both 3 and 4, it's replaced with "Bish-Bosh".

The purpose of the webpage is to let users choose which numbers to replace with "Bish" and "Bosh", and see the results on the screen.
